# RealEstateR

A real estate web app built using nextjs v11 and Bayut API

## Installation

### Prerequisite
- Node version 16.x.x

### Clone the repo
```bash
git clone https://github.com/raffyamoguis/movie-library-app.git
cd movie-library-app
```

### Run npm install

```bash
npm install
```

### Run development mode

```bash
npm run dev
```
